# Ultimate Sudoku — Figma Make snapshot

Source of truth: [`https://www.figma.com/make/CWUV24rOWs5OY24YK5JVoK/Ultimate-Sudoku`](https://www.figma.com/make/CWUV24rOWs5OY24YK5JVoK/Ultimate-Sudoku)

This folder contains a **local snapshot** of the Figma Make “Ultimate Sudoku” project:

- `source/`: Make-generated TypeScript/TSX/CSS/MD files
- `images/`: image assets referenced by the Make project
- `MANIFEST.json`: list of exported files

The snapshot exists to support deterministic audits and visual regression tests against the current RN/Expo implementation.
